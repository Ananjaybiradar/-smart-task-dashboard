# Smart Task Management Dashboard

This is a simple **Smart Task Management Dashboard** built using **React.js**.  
The application helps users manage daily tasks efficiently with features like task creation, editing, filtering, searching, and data persistence.

- Search tasks by title in real time
- Tasks are saved using **localStorage** (data remains after refresh)
- Clean and responsive UI
